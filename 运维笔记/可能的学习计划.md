## 🚀 高质量开源 Gin 博客项目推荐（持续维护）

### 1. **go-blog** by MuxiTech

- GitHub：https://github.com/muxi-tech/go-blog
- ⭐ Star：2.4k+
- 技术栈：
  - Gin + GORM + MySQL + JWT + Swagger
  - 配置分离，接口注释规范
- 特点：
  - 完整的博客系统（文章、标签、用户、登录）
  - 合理分层（router/controller/service/model）

✅ 学习推荐理由：

> 结构非常清晰，注释齐全，适合初学者学习项目分层和接口设计。

------

### 2. **ginblog** by JaysonFong

- GitHub：https://github.com/JaysonFong/ginblog
- ⭐ Star：600+
- 技术栈：
  - Gin + GORM + MySQL + JWT + Vue 前后端分离
- 特点：
  - 用户注册登录、文章发布、评论、分类管理
  - 后端接口对接前端 Vue 项目
- 教程作者还有完整视频教学（B 站）

✅ 学习推荐理由：

> 简单易懂，适合快速入门 Gin 构建 API 接口。

------

### 3. **gin-vue-blog**（全栈项目）

- GitHub：https://github.com/RaySongCoding/gin-vue-blog
- ⭐ Star：1k+
- 技术栈：
  - Gin 后端 + Vue3 前端
  - GORM + JWT + Casbin（权限）+ WebSocket + Swagger
- 特点：
  - 包含前后端完整实现
  - 支持文章、评论、标签、登录注册、权限系统等

✅ 学习推荐理由：

> 想深入了解“全栈+前后端分离”，这个是非常棒的实战案例。

------

### 4. **gin-admin**

- GitHub：https://github.com/LyricTian/gin-admin
- ⭐ Star：3.5k+
- 技术栈：
  - Gin + GORM + Casbin + JWT + Vue + gRPC
- 特点：
  - 模块化、RBAC 权限系统
  - 后台管理系统架构模板

✅ 学习推荐理由：

> 适合深入研究 Gin 架构、权限系统、模块解耦等大型项目结构。

------

## 🧠 建议你的学习路线

1. 从 **ginblog**（结构最轻）开始入门
2. 跟着 `go-blog` 研究接口风格与分层
3. 最后看 `gin-vue-blog` 或 `gin-admin` 学习大型架构、权限系统
